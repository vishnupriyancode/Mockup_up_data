CMD:
Single Command to Generate All Positive Scenarios:
python -m src.mockgen.cli --enhanced --output-format split --models Model_1_Positive Model_2_Positive

Alternative Commands for Different Scenarios:

Generate ALL Positive Scenarios (All Models):
python -m src.mockgen.cli --enhanced --output-format split --models Model_1_Positive Model_2_Positive

Generate ALL Scenarios (Positive + Negative + Exclusion):
python -m src.mockgen.cli --enhanced --output-format split

Generate Multiple Random Variations of Positive Scenarios:
python -m src.mockgen.cli --enhanced --output-format multiple --count 5 --models Model_1_Positive Model_2_Positive
